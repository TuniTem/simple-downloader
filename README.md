Hi there dreamer!

You have stumbled upon a streamlined mini-app that acts as a internet media downloader, inspired by cobalt.tools

Key features:
1. Download most media types from any site you can think of. You can see the full list of sites supportd [here](https://github.com/yt-dlp/yt-dlp/blob/master/supportedsites.md)
(Has been tested/verified on youtube, twitch, soundcloud, reddit - there's a small chance it wont be compatible with some of the other sites listed! [Please shoot me a dm on discord or open an issue](discord.gg/ZSsZxYhRRt))

2. Super simplified UI - there are many QOL features packed into this tiny window!
Very streamlined as well! After you copy a link, downloading media can litterally be a one click process!
<img width="958" height="265" alt="DreamloaderSS2" src="https://github.com/user-attachments/assets/c66fdc39-ee08-49b6-b15a-62bfb5a92713" />

3. Buuut it can be a lil more complicated if you need the settings, the choice is yours!
I do plan to add more advanced settings in the future! I have many many ideas c:

4. Download queue - input as many links as you like, whenever you like!
You can also provide a path to a text file containing a list of links, either comma seperated or each on their own line

<img width="977" height="490" alt="image" src="https://github.com/user-attachments/assets/e635b7de-1a97-44d1-9d18-317ef0668646" />

5. Playlist downloading - automatically sort downloaded playlists into their own files, with titles retrived and automatically generated!

6. Lots of polish that you might not expect in an app! But you can turn it off if you dont like it.

7. Smooth download predictions that adjust to your computer's specs and internet speed!

8. Text flags to automatically name downloaded files! here's the full list:
```
[title] : The title of the peice of media

[author] : Whoever created the peice of media!

[date] : The date the peice of media was created in MM/DD/YYYY

[date_d] : The date the peice of media was crated in DD/MM/YY

[year] : the year the peice of media was created

[duration] : The length of the peice of media

[resolution] : The resolution of the media in LxH (ex: 1920x1080)

[quality] : If its a video, uses the "P" resolution (like 1080p), if its audio tells you the bitrate in kb/s!

[playlist_number] : the index number of a peice of media if its in a playlist. Is ignored if it is not a playlist.
```
